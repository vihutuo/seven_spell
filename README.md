The same  of 7 letters  is given to all users every 60 seconds.
Create as many 3+ letter words as you can using the 7 letters given to you before the timer rounds out.
A score board of all users is displayed after time ends

Seven Spell was built as a class presentation project (2024) of Little Star Higher Secondary School, Nagaland by students of Class 12.
